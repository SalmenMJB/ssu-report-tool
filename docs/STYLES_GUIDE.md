# Guide des Styles - Rapport SSU

## Titres

### Titre de chapitre (H1)
- **Couleur** : Bleu ciel (#0099CC)
- **Taille** : 28pt
- **Gras** : Oui
- **Alignement** : Centré
- **Espacement** : 12pt avant et après

**Exemple** : "Service infirmier"

### Titre de section (H2)
- **Couleur** : Bleu ciel (#0099CC)
- **Taille** : 16pt
- **Gras** : Oui
- **Alignement** : Gauche
- **Espacement** : 12pt avant, 6pt après

**Exemple** : "Faits marquants de l'année"

### Sous-titre (H3)
- **Couleur** : Noir (#333333)
- **Taille** : 13pt
- **Gras** : Oui
- **Italic** : Non
- **Alignement** : Gauche

**Exemple** : "Formation et encadrement"

## Texte courant

- **Police** : Calibri 11pt
- **Couleur** : Noir (#333333)
- **Interligne** : 1.15
- **Alignement** : Justifié

## Encadrés

### Encadré normal
- **Bordure** : Bleu ciel 2pt
- **Fond** : Blanc
- **Remplissage** : 6pt
- **Utilisation** : Texte important

### Encadré processus
- **Bordure** : Bleu ciel, arrondie
- **Connexions** : Flèches grises entre encadrés
- **Utilisation** : Diagrammes de flux

## Tableaux

### En-têtes
- **Couleur de fond** : Bleu ciel (#0099CC)
- **Couleur du texte** : Blanc
- **Police** : Calibri 11pt gras

### Lignes alternées
- **Ligne paire** : Blanc
- **Ligne impaire** : Gris clair (#F5F5F5)

### Bordures
- **Couleur** : Gris (#CCCCCC)
- **Largeur** : 1pt

## Mots clés

- **Couleur** : Noir (#333333)
- **Gras** : Oui
- **Utilisation** : Nombres, chiffres importants dans le texte

**Exemple** : "Cette année, **22 sessions** réalisées"
